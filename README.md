# Advanced CSS and Sass: Course Projects

This repository contains my progress and code from the comprehensive course **"The most advanced and modern CSS course on the internet"** by Jonas Schmedtmann. The goal of this project is to master professional CSS workflows, including architecture, responsive design, and advanced layout techniques.

## 🚀 What I've Learned
* **Three Pillars of CSS:** How CSS works behind the scenes (cascade, specificity, inheritance).
* **Sass (SCSS):** Using variables, mixins, nesting, and the 7-1 architecture.
* **Advanced Layouts:** Deep dive into Flexbox and CSS Grid.
* **Responsive Design:** Desktop-first and mobile-first approaches with media queries.
* **Build Tools:** Simple build processes using NPM scripts to compile Sass and autoprefix CSS.

## 📁 Projects Included
The course is built around three major projects, each focusing on different CSS aspects:

1.  **Natours** – Focuses on advanced CSS animations, Sass architecture, and float-based layouts (transitioning to modern tools).
2.  **Trillo** – An all-Flexbox project focusing on component-based design.
3.  **Nexter** – A complex layout built entirely with CSS Grid.

## 🛠️ Built With
* HTML5
* CSS3 (Advanced features)
* Sass / SCSS
* NPM (Build scripts)
